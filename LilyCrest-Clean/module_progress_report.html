Priority Recommendations

🔴 Critical — Blocks Real-World Use

1
Reservation Module
Tenants can't view their reservation status, history, or move-in/out details. Entire module is missing.
2
Admin Maintenance Status Update API
No endpoint for admin to update request status. Push notification function is orphaned. Tenants never know when their request is resolved.
3
Push Notification Integration
Infrastructure exists (token registration + Expo Push API) but no notifications are actually sent. Tenants miss important updates.

🟡 Important — Impacts User Experience

4
Monthly Billing Statement Notification
No push/email when new bills are released. Tenants must manually check for new bills.
5
Announcement Acknowledgment
No way to confirm tenants have read important notices or policy updates.
6
Move-Out Reminders
No lease-end date tracking in user model. No automated reminder notifications.

🟢 Nice to Have — Polish

7
Email Verification
Firebase supports it but the flow is not implemented. Low risk for tenant-only app with admin-provisioned accounts.
8
Maintenance Status Timeline
Reopen history is stored but no visual timeline UI. Would improve request tracking experience.
9
Tenant Spending Dashboard
Monthly payment summaries and spending trends to help tenants track expenses.
